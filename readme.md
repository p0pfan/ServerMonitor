### Server Monitor Web Application

---

#### 1. Introduction
This web Application can be used to monitor the CPU and Memory usage **Linux server**. It has a Connect page and a Monitor page.  

![Connect Page](http://7xrn7f.com1.z0.glb.clouddn.com/16-6-21/20821681.jpg)  
![Monitor Page](http://7xrn7f.com1.z0.glb.clouddn.com/16-6-21/70555586.jpg)  
![DataBase](http://7xrn7f.com1.z0.glb.clouddn.com/16-6-21/30188260.jpg)  

- Demo:localhost:8080/servermonitor/status/connect
- **Make sure the remote server can be connected by using SSH**

---

#### 2. Main Feature

- Login Page
    - Server IP, username, password is needed to connect by using `jsch.jar`;
    - If server is connected, it will jump to the monitor page `localhost:8080/servermonitor/status/monitor`, other wise it will stay in this page. 


- Monitor Page
    - "Disconnect" button is used to disconnect the server.
    - "REFRESH" checkbox is used to refresh the server usage every 1sec.
    - Two charts show the usage change.

- Metric Collection job 
    - the web server collects the CPU and Memory usage of remote server from the front end every 30sec by using quartz. Data is stored into MySQL.
